# Blender-Bonsai-IfcOpenShell-Sverchok Claude Skill Package

A comprehensive Claude Skill Package for AEC (Architecture, Engineering, Construction) technologies.

## Technologies Covered
- **Blender** - 3D modeling, Python API, addon development
- **Bonsai** - BIM addon for Blender (formerly BlenderBIM)
- **IfcOpenShell** - IFC geometry engine and Python library
- **Sverchok** - Visual programming addon for Blender (planned)
